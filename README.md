# Queue Reminiscence

A free queue-board system project.

## Engineering log

Project execution is tracked under `.hermes/`:

- `.hermes/engineering-journal.md` — append-only project journal: decisions, blockers, evidence, and session notes.
- `.hermes/time-log.csv` — append-only time ledger for measuring time spent by work phase.
- `.hermes/time-metrics.md` — definitions and workflow for consistent time tracking.

Primary metric: **time taken**, grouped by phase such as planning, implementation, testing, debugging, documentation, and review/deployment.
